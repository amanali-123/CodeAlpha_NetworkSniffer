# Basic Network Sniffer 🔍

## Project Description
A Python-based network sniffer that captures and analyzes network packets in real-time.

## Built With
- Python
- Scapy Library
- Npcap

## Features
- Captures live network packets
- Shows Source & Destination IP
- Identifies TCP, UDP, ICMP protocols
- Displays port numbers

## How to Run
1. Install Python
2. Run: pip install scapy
3. Install Npcap
4. Run: python sniffer.py
